Explicación de un código en C


    #include <stdio.h>
    int main()
    {
        int edad;
        printf("¿Cuántos años tienes? ");
        scanf("%d", &edad);
        if(edad >= 18)
        {
            printf("Eres mayor de edad.\n");
        }
        else
        {
            printf("Eres menor de edad.\n");
        }
        return 0;
    }

Se declara la biblioteca: #include <stdio.h>

Se declara una variable de tipo entero llamda edad: int edad;

Se imprime un mensaje en pantalla: printf("¿Cuántos años tienes? ");

Se lee un número entero desde el teclado y se guarda en la variable edad: scanf("%d", &edad);

Se evalúa si la edad es mayor o igual a 18: if(edad >= 18)

Si la condición es verdadera, se imprime un mensaje indicando que la persona es mayor de edad: printf("Eres mayor de edad.\n");

Si la condición es falsa, se imprime un mensaje indicando que la persona es menor de edad: printf("Eres menor de edad.\n");

Finalmente, el programa retorna 0 para indicar que terminó correctamente: return 0;
